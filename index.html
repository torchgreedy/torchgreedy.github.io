  • Designed and implemented a custom Transformer-based architecture featuring dedicated EEG and eye-tracking encoders, Cross-Modal Attention mechanism, and Feature Importance Modules

Cross-Modal Transformer with Adversarial Domain Adaptation

March 2025 - August 2025

Project Overview

This research project focused on developing a novel deep learning approach for emotion recognition using multimodal data from EEG and eye-tracking. The key innovation was the implementation of adversarial domain adaptation to address the challenge of inter-subject variability in physiological signals.

Key Features

  • Custom Transformer architecture with cross-modal attention mechanism
  • Adversarial domain adaptation through gradient reversal layer
  • Subject-specific normalization layers for improved generalization
  • Feature importance visualization to interpret model decisions
  • Rigorous Leave-One-Subject-Out evaluation on the SEED-V dataset across 16 subjects

Project Overview
